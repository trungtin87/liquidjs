Sử dụng trong Express.js

LiquidJS tương thích với các công cụ mẫu express. Bạn có thể đặt phiên bản liquidjs cho tùy chọn công cụ xem:

var { Liquid } = require('liquidjs');
var engine = new Liquid();

// đăng ký công cụ liquid
app.engine('liquid', engine.express()); 
app.set('views', './views');            // chỉ định thư mục views
app.set('view engine', 'liquid');       // đặt liquid làm mặc định
Working Demo

Đây là một bản demo hoạt động cho việc sử dụng LiquidJS trong Express.js: liquidjs/demo/express/.

Tra cứu mẫu

Tùy chọn root sẽ tiếp tục hoạt động như là gốc của các mẫu, như bạn có thể thấy trong Hiển thị một tệp mẫu. Ngoài ra, tùy chọn views trong express.js (như được hiển thị ở trên) cũng sẽ được tôn trọng. Giả sử bạn có một thư mục mẫu như:

.
├── views1/
│ └── hello.liquid
└── views2/
  └── world.liquid

Và bạn đang đặt thư mục gốc của mẫu cho liquidjs là views1 và cho expressjs là views2:

var { Liquid } = require('liquidjs');
var engine = new Liquid({
    root: './views1/'
});

app.engine('liquid', engine.express()); 
app.set('views', './views2');            // chỉ định thư mục views
app.set('view engine', 'liquid');       // đặt liquid làm mặc định

Cả hello.liquid và world.liquid đều có thể được giải quyết và hiển thị:

res.render('hello')
res.render('world')

Bộ nhớ đệm

Chỉ cần đặt tùy chọn bộ đệm thành true sẽ bật bộ nhớ đệm mẫu, như được giải thích trong Bộ nhớ đệm. Nên bật bộ nhớ đệm trong môi trường sản xuất, có thể được thực hiện bằng cách:

var { Liquid } = require('liquidjs');
var engine = new Liquid({
    cache: process.env.NODE_ENV === 'production'
});
